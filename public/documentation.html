My Flix Documentation

This project is a web application that will provide users with information about movies, directors, and genres. Users will also be able to sign up, log their personal information, and create a list of their favorites.

Request Endpoint URL Method Type Body Data Format Response Data Format Description
Get data on all movies /movies GET N/A A JSON object containing data on all the movies currently on the server This request will respond with a list of all movies on the server, including all of their data such as title, description, genre, director, etc.
Get data on a specific movie /movies/:title GET N/A A JSON object containing data on the movie title selected This request will respond with a list of data on the movie selected, listing data such as title, description, genre, director, etc.
Get data on a specific genre /movies/genre/:genreName GET N/A A JSON object containing data on the movie genre selected This request will respond with a list of data on the genre selected, listing the genre and a description of it.
Get data on a specific director /movies/directors/:directorName GET N/A A JSON object containing data on the director selected This request will respond with a list of data on the director selected, listing their name, birth year, and a short bio.
Add a new user /users POST A JSON object formatted like so:
{
"name": user's name
"favoriteMovies": []
} 		A JSON object containing the information entered, but now with a unique user id number. This request will allow the user to enter a name and be given a unique id number for their account
Add a new favorite movie /users/:id/:movieTitle POST N/A A text response stating the movie has been added to the user's favorites This request will allow the user to add movies to the "favoriteMovies" array linked to their personal id
Update a username /users/:id PUT A JSON object formatted like so:
{
"name": user's new username
"favoriteMovies": []
} 		A JSON object containing their account information, but now with their updated name. This request will allow the user to enter a name that will replace their current username
Remove a favorite movie /users/:id/:movieTitle DELETE N/A A text response stating the movie has been removed from the user's favorites This request will allow the user to remove specific movies from their "favoriteMovies" array
Remove a user /users/:id DELETE N/A A text response stating the user has been deleted This request will allow the user to delete their user account
